En este ejercicio debemos crear una clase "Persona" con las siguientes propiedades: _idPersona (number), _nombre (string), _apellido (string), _edad (number), contador de personas. Recordemos de usar el método constructor para ingresar los futuros datos. Además, debemos crear las funciones get y set para cada uno de las propiedades.
